# Roadmap

- [x] Core SDK with event-driven API (`speechStart`, `pause`, `turnEnd`, `interrupt`)
- [x] EnergyVAD baseline classifier
- [x] Published to npm as [`@utterance/core`](https://www.npmjs.com/package/@utterance/core)
- [x] Documentation site and interactive playground
- [ ] Train and ship the semantic endpointing model (ONNX)
- [ ] ONNX Runtime Web inference integration
- [ ] React hook (`useUtterance`)
- [ ] CI/CD pipeline with automated npm publishing
- [ ] Framework adapters (Vue, Svelte)
- [ ] Voice SDK integration examples (LiveKit, Daily, Twilio)

Have a feature in mind? [Open an issue](https://github.com/nizh0/Utterance/issues) or [join the Discord](https://discord.gg/kb4zMHNtEV).
